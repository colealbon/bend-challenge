# bend-challenge
a project to assess competence

# installation
```
git clone https://github.com/colealbon/bend-challenge.git
cd bend-challenge;
npm install;
# (modify config/options.js if needed)
npm test;
npm start;
```

# usage   
navigate to localhost:3000 (or port settings from config/options.js)
